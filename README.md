# Team Task Manager

## Overview

Team Task Manager is a full-stack collaborative web application designed to help teams manage projects, assign tasks, and track work progress efficiently.

The application supports role-based access control where Admins can manage projects and team members, while Members can view and update assigned tasks.

This project was built as a simplified collaborative productivity platform inspired by tools like Trello and Asana.

---

# Features

## User Authentication
- Secure Signup and Login
- JWT-based Authentication
- Role Selection during Signup (Admin / Member)

## Role-Based Access Control

### Admin
- Create and manage projects
- Add and remove members
- Assign tasks to team members
- View all projects and tasks

### Member
- View assigned projects
- Update task status
- Track project progress
